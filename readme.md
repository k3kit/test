# AutoDriveSaratov - Премиум автомобили

Сайт автосалона премиальных автомобилей с современным дизайном и интерактивными элементами.
## Требования для запуска проекта

### Node.js

Для запуска проекта необходима установленная **Node.js** версии 16 или выше.

```bash
# Проверьте версию Node.js
node --version

## 🚀 Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 
- Tailwind
- Vite

## ✨ Функционал

- Адаптивный дизайн
- Анимация элементов при скролле
- Модальное окно с деталями автомобилей
- Валидация формы обратной связи
- Карусель изображений
- Эффекты при наведении на карточки

## 🛠️ Установка и запуск

```bash
# Клонирование репозитория
git clone <url-репозитория>

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
